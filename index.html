All files are .tar format. Use with Samsung Odin3 on Windows to flash to the AP slot. Disable Auto Reboot before flashing.
Installation Guide
1
Flash vbmeta Disabler
Boot into Download Mode (hold Volume Down + Power → release, then press Volume Up). Launch Odin3 and disable Auto Reboot in the Options tab. Load vbmeta_disabled.tar into the AP slot and click Start.
2
Flash TWRP Image & Factory Reset
When done, reboot into factory recovery by holding Volume Up + Power. Factory reset the phone, then power it off. Boot back to Download Mode and plug into PC. In Odin, load twrp-3.6.0_afaneh92-a11q.tar into the AP slot with Auto Reboot disabled. Click Start.
3
Run Multidisabler in TWRP
After flashing, boot into TWRP recovery by holding Volume Up + Power. Go to Advanced → Terminal and type multidisabler, then press Enter. Go back and select Reboot → System.

Useful Commands
# Check device and bootloader status via ADB
adb shell getprop ro.bootloader
adb shell getprop ro.serialno

# Reboot into bootloader/download mode
adb reboot bootloader

# Check if TWRP is installed (from ADB in recovery)
adb shell cat /etc/recovery.fstab

# Sideload Magisk or other ZIPs from TWRP
adb sideload magisk.zip

# Pull Nandroid backup to PC
adb pull /data/media/0/TWRP/BACKUPS/
